# Règles de consolidation éditoriale

| Cas | Règle à appliquer |
|-----|-------------------|
| **Deux experts disent la même chose** | Fusionner en une seule formulation. Choisir la plus précise. Ne pas doubler. |
| **Deux experts se contredisent** | Ne pas trancher soi-même. Formuler la contradiction comme une décision ouverte pour le CPO avec les deux positions clairement énoncées. |
| **Un expert sort du périmètre** | Filtrer. Ne pas intégrer dans la spec. Si pertinent pour le futur, consigner dans le document de transparence. |
| **Un retour est trop générique** | Reformuler en règle produit concrète ou demander une précision avant intégration. |
| **Un retour contredit une décision déjà prise dans le brief** | Signaler la contradiction au CPO. Ne pas effacer la décision du brief sans arbitrage explicite. |
| **Un retour concerne une version future** | Déplacer dans la section 4.3 « Reporté à une version future ». Ne pas l'inclure dans le scope actuel. |
